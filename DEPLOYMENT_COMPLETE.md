### Full Checklist (30 minutes)
See: [`PRODUCTION_DEPLOYMENT_CHECKLIST.md`](./PRODUCTION_DEPLOYMENT_CHECKLIST.md)

### Step-by-Step Guide
1. **Configure Environment Variables** (5 min)
   - Go to Vercel dashboard
   - Add all required environment variables
   - Save configuration

2. **Set Up Supabase** (10 min)
   - Create Supabase project
   - Run migrations
   - Configure RLS policies
   - Create indexes

3. **Deploy** (5 min)
   - Automatic: Code on main → Vercel auto-deploys
   - Manual: Run `vercel --prod`

4. **Verify** (5 min)
   - Check app loads
   - Test login
   - Test admin dashboard
   - Check for errors

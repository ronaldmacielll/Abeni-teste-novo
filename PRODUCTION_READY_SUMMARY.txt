🚀 DEPLOYMENT STEPS
═══════════════════════════════════════════════════════════════════════════════

  1. Configure Environment Variables (Vercel)
     - NEXT_PUBLIC_SUPABASE_URL
     - NEXT_PUBLIC_SUPABASE_ANON_KEY
     - SUPABASE_SERVICE_ROLE_KEY
     - INSTAGRAM_ENCRYPTION_KEY
     - INSTAGRAM_VAULT_URL
     - INSTAGRAM_SYNC_FREQUENCY_MINUTES=5
     - CLICKUP_API_KEY
     - WEBHOOK_SECRET
     - NODE_ENV=production

  2. Set Up Supabase
     - Create project
     - Run migrations
     - Configure RLS policies
     - Create indexes

  3. Deploy
     - Automatic: Code on main → Vercel auto-deploys
     - Manual: vercel --prod

  4. Verify
     - Check app loads
     - Test login
     - Test admin dashboard
     - Check for errors
